Full trinn-for-trinn veiledning

Installer chatbot og adminpanel på 15–25 minutter

Systemet inkluderer: chatbot på nettstedet, lagring av samtaler i JSON, adminpanel for å se alle dialoger og sende svar via Telegram.

6

Підключіть чат-бот до вашої сторінки

Додайте цей рядок безпосередньо перед закриваючим тегом </body> у вашому index.html (або index.php):

<script src="/marit/marit-chat.js" defer></script>
1

Forbered serveren

Krav

  • ✓ PHP 8.1+
  • ✓ Skrivetillatelser på mapper (chmod 755–775)
  • ✓ HTTPS (sterkt anbefalt)
  • ✓ curl, json, session

Anbefalt mappestruktur 

marit/
├── config.php
├── index.html (eller index.php)
├── marit-chat.js
├── marit-bot.php
├── marit-get-messages.php
├── marit-telegram-webhook.php
├── marit-admin.php (adminpanel)
├── conversations/ ← samtaler lagres her
└── log/ ← logger og rate-limit
2

Opprett og konfigurer config.php

Viktig! Denne filen inneholder alle nøkler. Aldri last den opp til offentlig Git.

Endre ADMIN_PASSWORD, XAI_API_KEY og Telegram-innstillinger.

3

Last opp alle filer

Hovedfiler

  • marit-chat.js
  • marit-bot.php
  • marit-get-messages.php
  • marit-telegram-webhook.php
  • marit-admin.php

Adminpanel

marit-admin.php — én fil for å se alle samtaler

Frontend

index.html + marit-chat.js (lastes med defer)

4

Sett riktige rettigheter

# Mapper
chmod 755 conversations log
# Filer
chmod 644 *.php *.js *.html config.php
# JSON-chatfiler (etter opprettelse)
chmod 644 conversations/s_*.json
På shared hosting bruk filbehandler (cPanel / DirectAdmin) → Permissions → 755 for mapper, 644 for filer.
5

Test at alt fungerer

  1. 1.
    Åpne nettstedet → chatboten skal dukke opp nede til høyre
  2. 2.
    Skriv en melding → sjekk Telegram (du skal få varsel)
  3. 3.
    Åpne /marit/marit-admin.php → skriv inn passord → du ser liste over sesjoner
  4. 4.
    Klikk på en sesjon → hele samtalen skal lastes inn
Ferdig! 🎉

Systemet fungerer fullt ut: chat på nettstedet + adminpanel for å se alle dialoger
