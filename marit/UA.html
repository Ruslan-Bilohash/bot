Повна покрокова інструкція

Встановіть чат-бота та адмін-панель за 15–25 хвилин

Система включає: чат-бота на сайті, збереження розмов у JSON, адмін-панель для перегляду всіх діалогів та відправку відповідей через Telegram.

<script src="/marit/marit-chat.js" defer></script>
1

Підготовка сервера

Вимоги

  • ✓ PHP 8.1+
  • ✓ Права на запис у папки (chmod 755–775)
  • ✓ HTTPS (дуже бажано)
  • ✓ curl, json, session

Рекомендована структура 

marit/
├── config.php
├── index.html               (або index.php)
├── marit-chat.js
├── marit-bot.php
├── marit-get-messages.php
├── marit-telegram-webhook.php
├── marit-admin.php          (адмінка)
├── conversations/           ← чати зберігаються тут
└── log/                     ← логи та rate-limit
2

Створіть та налаштуйте config.php

Важливо! Цей файл містить усі ключі. Ніколи не завантажуйте його в публічний Git.

Змініть ADMIN_PASSWORD, XAI_API_KEY та Telegram налаштування.

3

Завантажте всі файли

Основні

  • marit-chat.js
  • marit-bot.php
  • marit-get-messages.php
  • marit-telegram-webhook.php
  • marit-admin.php

Адмін-панель

marit-admin.php — єдиний файл для перегляду всіх розмов

Фронтенд

index.html + marit-chat.js (підключається defer)

4

Встановіть права доступу

# Папки
chmod 755 conversations log

# Файли
chmod 644 *.php *.js *.html config.php

# JSON-файли чатів (після появи)
chmod 644 conversations/s_*.json
На shared-хостингу використовуйте файлову панель (cPanel / DirectAdmin) → Права → 755 для папок, 644 для файлів.
5

Перевірка роботи

  1. 1.
    Відкрийте сайт → чат-бот повинен з'явитись в правому нижньому куті
  2. 2.
    Напишіть повідомлення → перевірте Telegram (має прийти сповіщення)
  3. 3.
    Відкрийте /marit/marit-admin.php → введіть пароль → побачите список сесій
  4. 4.
    Клікніть на сесію → має завантажитись повна розмова
Готово! 🎉
