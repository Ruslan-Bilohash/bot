# 💆‍♀️ Body & Face Studio Chat Bot

**Розумний чат-бот для масажного салону Body & Face Studio (Вільнюс)**

Сучасний, адаптивний та професійний чат-бот, який допомагає відвідувачам швидко записатися на масаж. Працює 24/7, збирає контакти, вид масажу, тривалість і відправляє всі заявки вам у Telegram.

---

## ✨ Основні можливості

- 💆‍♀️ Повністю українською мовою (з підтримкою усіх мов світу)
- 🏠 Представляє весь масажний салон **Body & Face Studio**, а не одну людину
- 📱 Адаптивний та сучасний дизайн (відмінно виглядає на телефонах і комп’ютерах)
- 🔒 Підвищена безпека: rate-limit, валідація сесій, security headers
- 📨 Миттєві сповіщення в Telegram (нові клієнти + всі відповіді)
- 💾 Автоматичне збереження історії розмов
- ⚙️ Легке налаштування через `settings.php` (кольори, контакти, промпт)
- 🚀 Швидка інтеграція на будь-який сайт

---

## 🛠 Технології

- **Frontend**: Vanilla JavaScript + сучасний CSS
- **Backend**: PHP 8+
- **AI**: Grok (xAI) — потужна модель 2026 року
- **Сповіщення**: Telegram Bot API
- **Зберігання**: JSON-файли (без бази даних)
